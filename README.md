## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher) or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/OMPRATIK/Sparklehood.git
   cd Sparklehood
   ```
2. Install dependencies
   ```bash
   npm install
   ```
### Run application

```bash
   npm run dev
```
